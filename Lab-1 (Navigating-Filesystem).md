# Lab 1: Navigating the Filesystem

## Objectives
- Understand the basic structure of a Linux filesystem
- Determine the current working directory
- List files and directories
- Navigate the filesystem using core commands

## Commands Practiced

| Command | Purpose |
|---|---|
| pwd | Print current working directory |
| ls | List files/folders in current directory |
| ls -la | List all files (including hidden) with details |
| cd / | Navigate to root directory |
| cd /etc | Navigate to a specific absolute path |
| cd .. | Move up one directory (relative path) |
| cd ~ | Jump to home directory |
| cd - | Return to previous directory |

## Key Concepts
- *Absolute path*: full path from root (/), works from anywhere
- *Relative path*: path relative to current location, doesn't start with /
- . = current directory, .. = parent directory
